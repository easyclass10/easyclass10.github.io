Calculadora de Derivadas Interactiva

Analiza funciones con cálculo diferencial usando SageMath 📊

Volver
Estado del núcleo de cálculo: Cargando...

Haz clic en los bloques de código para ejecutarlos

Ejemplo: Esta herramienta analiza la función f(x) = 2x³ - 3x² - 12x + 1
Puedes modificar el código para analizar cualquier función que necesites.

1. Primera Derivada: Crecimiento y Decrecimiento

Calcula la primera derivada y encuentra dónde la función crece o decrece.

Definir función y calcular derivada:
sage: f = 2*x^3-3*x^2-12*x+1
sage: df = diff(f,x)
sage: df

sage: solve(df,x)
¿Dónde crece la función?
sage: solve(df>0,x)

2. Extremos Relativos (Máximos y Mínimos)

Usa la segunda derivada para clasificar los puntos críticos.

sage: ddf = diff(df,x)
sage: ddf

sage: [ddf.subs({x:-1}), ddf.subs({x:2})]

3. Segunda Derivada: Concavidad y Convexidad

Determina dónde la función es cóncava hacia arriba o hacia abajo.

sage: solve(ddf>0,x)

4. Puntos de Inflexión

Encuentra dónde cambia la concavidad de la función.

sage: solve(ddf,x)

sage: diff(ddf,x)

5. Visualización de la Función

Genera la gráfica de la función para visualizar su comportamiento.

sage: plot(f, (-5,5), ymax=20, ymin=-20)

¿Cómo usar esta herramienta?

  1. Espera a que el núcleo de cálculo esté listo
  2. Haz clic en cualquier bloque de código gris para ejecutarlo
  3. Los resultados aparecerán debajo de cada bloque
  4. Puedes modificar el código para analizar otras funciones
  5. Ejecuta los bloques en orden para un análisis completo
